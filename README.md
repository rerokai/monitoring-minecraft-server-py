Бэкенд на FastAPI асинхронно опрашивает Prometheus по PromQL-запросам и через REST‑эндпоинты (/api/metrics, /api/minecraft/metrics, /api/*/range) мгновенно отдаёт системные (CPU, RAM, диск, swap, uptime и др.) и игровые метрики (TPS, онлайн, чанки, энтити, размер мира), используя httpx для асинхронных вызовов и requests для range-запросов, с поддержкой CORS и настройкой через переменные окружения.
